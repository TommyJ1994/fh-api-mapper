Service Setup Needed

  1. Make the service public
    In the Service Details page of the newly created API Mapper service, in the Access Control section, check Make this Service Public to all Projects and Services and click Save Service.

  2. Upgrade the database of the service
    The API Mapper service needs a dedicated database. To perform this task, the service needs to be fully deployed and running.

    1. In the Data Browser section of the newly created service, click the Upgrade Database button in the top right corner, and confirm by clicking Upgrade Now.

    2. Redeploy the service by clicking Deploy Cloud App in the Deploy section.
